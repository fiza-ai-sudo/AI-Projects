# Chatbot Project

## Description
This is a simple AI chatbot built using Python.
It can greet users, answer basic questions, and do small talk.

## Features
- Greeting (Hi / Hello)
- Question and Answer
- Small Talk
- Simple chatbot logic

## Technologies Used
- Python
- Flask
- Rule-based logic

## How to Run (Google Colab)
1. Open the notebook in Google Colab
2. Run all cells
3. Enter your message to chat with the bot
